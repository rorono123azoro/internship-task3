# Internship Task 3: React To-Do List

## Overview
A functional, component-driven To-Do list application built using the React framework (scaffolded with Vite). It demonstrates fundamental frontend concepts like component composition, state management, and side-effect handling.

## Key Features
- **Task Management**: Seamlessly add, edit, or delete task items on the fly.
- **State Handling**: Uses React Hooks (e.g., `useState`, `useEffect`) for localized state.
- **Component Architecture**: Structured cleanly into reusable UI containers and specific feature sets in an industry-standard directory layout.
- **Fast Tooling**: Bundled exceptionally fast using Vite.

## Technologies Used
- React (18+)
- TypeScript
- Vite
- CSS Modules / Vanilla CSS

## Setup Instructions
1. Ensure [Node.js](https://nodejs.org/) is installed on your computer.
2. Clone the repository and navigate into the folder:
   ```bash
   cd task3
   ```
3. Install frontend dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Visit the Localhost URL provided in the terminal (typically `http://localhost:5173`) in your browser to interact with the application.
